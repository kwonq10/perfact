スキマ - プライバシーポリシー

Privacy Policy

はじめに

スキマ（以下「本サービス」）は、Googleカレンダーと連携してユーザーの空き時間を検索するWebアプリケーションです。本プライバシーポリシーは、本サービスが取得する情報とその取り扱いについて説明します。

取得するGoogleユーザーデータ

本サービスは、ユーザーがGoogleアカウントでログインした際に以下の情報にアクセスします。

  • Googleカレンダーの予定情報（タイトル、開始時刻、終了時刻）
  • Googleカレンダーの一覧情報（カレンダー名、カレンダーID）

メールアドレスやその他の個人情報は取得しません。

情報の使用目的

取得した情報は以下の目的にのみ使用します。

  • ユーザーの指定した期間における空き時間の計算と表示
  • ユーザーが対象カレンダーを選択できるようにするため

上記以外の目的には一切使用しません。

情報の保存・共有について

本サービスは取得したカレンダーデータをサーバーに保存しません。すべての処理はユーザーのブラウザ上で行われます。

取得した情報を第三者に販売・提供・共有することは一切ありません。

取得した情報を広告配信に使用することは一切ありません。

取得した情報をAIモデルの学習に使用することは一切ありません。

本サービスはGoogle Workspace API User Data PolicyのLimited Use requirementsに準拠しています。

ブラウザへの保存情報（localStorage）

本サービスはユーザーのブラウザ（localStorage）に以下の情報を保存します。

  • gtoken：Google APIアクセストークン
  • gtoken_expiry：アクセストークンの有効期限（取得から約1時間で自動無効化）
  • install_dismissed：PWAインストール案内の非表示状態

これらの情報はサーバーへ送信されることはありません。

Googleアカウントの権限について

本サービスが使用するGoogleの権限は以下の通りです。

  • https://www.googleapis.com/auth/calendar.events.readonly：カレンダーの予定を読み取るための権限（読み取りのみ）
  • https://www.googleapis.com/auth/calendar.calendarlist.readonly：カレンダーの一覧を読み取るための権限（読み取りのみ）

本サービスはカレンダーへの書き込み・編集・削除は一切行いません。

アクセス権限の取り消し

ユーザーはいつでも以下のページからスキマへのアクセス権限を取り消すことができます。

https://myaccount.google.com/permissions

Cookieについて

本サービスは分析・広告目的のCookieを使用しません。ログイン状態の維持のみにブラウザのlocalStorageを使用します。
